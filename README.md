### Install Dependencies

Install [Nestjs CLI](https://docs.nestjs.com/cli/usages)

```bash
# npm
npm i -g @nestjs/cli
# yarn
yarn add -g @nestjs/cli
```

Install the dependencies for the Nest application:

```bash
# npm
npm install
# yarn
yarn install
```
